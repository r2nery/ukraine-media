# Monitoring the Russo-Ukranian War Through Media

## Reference
This repository is a refactored version of the reproduction package for Resonant Journalism in the Russo-Ukrainian War: A Topic Modeling Approach to Key-Point Detection, the thesis for my BSc in Economics (Universidade de Bras√≠lia - Brazil)

## Features
### Working scrapers for 11 news outlets:
- ABC
- AP
- CBS
- CNN
- DailyMail
- Express
- Fox
- Guardian
- HuffPost
- Mirror
- NY Times
- Reuters
- RT

### Topic Analysis
- (Barron et al. 2018) Implementation of latent Dirichlet allocation (LDA)
- KLD-Based measures of Novelty, Resonance and Transience
